# Sales Dashboard using Power BI

## Project Overview
This project is an interactive sales dashboard created using Power BI to analyze:
- Sales trends
- Regional performance
- Top-selling products
- Profit analysis

## Features
- KPI Cards
- Sales Trend Analysis
- Product-wise Sales
- Region-wise Performance
- Interactive Slicers

## Tools Used
- Power BI
- Excel
- GitHub
